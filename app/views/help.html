'Files' is one of the four main categories of project resources. Unify does not host files, but lets you share links to files. This way you can keep working with the tools you love, like Dropbox, Google Docs and Evernote.

File types

You can add any link you want as a file. Unify will try to recognize the file type and display a preview.

Unify currently recognizes files from Dropbox, Google Drive, Google Maps, Youtube and Evernote. We are working on supporting more platforms and file types.

Adding a file

To add a file, you just have to provide us with a (public) link to the file.

Obtaining a public link can be different for each file provider. Here's how to do it for a few major providers:

Dropbox

With the Dropbox web application, you can obtain a public link by clicking the Share button. Here's a screenshot of the Share button on the Image Previewer:

If you have installed the Dropbox software on your computer, you can also create a public link by choosing 'Share Dropbox Link' or 'Copy Public Link' from the right-click menu.
